


# LLM Spot v0.1.0-Pre-aplha


## Installation

### Requirements
- Node.js
- npm (or another package manager)
- Python 3
- Pipenv

### Steps
1. Clone the repository.
2. Navigate to `./backend` and run:
   ```bash
   pipenv install
   python ./api.py
   ```
3. In the root directory, start the frontend:
	```bash
	npm start
	```
